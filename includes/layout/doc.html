Documentation

L'application web

Le but de cette application web est l'analyse de données géothermique. À travers une interface claire et simple d'utilisation, vous pourrez analyser un ensemble de donnée préalablement acquise, grâce à différents outils (graphique, tableau de statistique)

Les vues 3D

Les vue 3D sont des représentations schématique des différentes installations réalisées par le département GTE. Ces vues permette de définir au mieux les capteurs dont ont veux étudier les données.

Se déplacer

Pour se déplacer dans la vue 3D, vous devez utiliser la souris. Le clic gauche permet de faire tourner l'ensemble de la vue 3D autour du point central. Le clic droit quant à lui permet de déplacer la vue latéralement.

Zoomer

Pour zoomer dans la vue 3D, vous devez utiliser la molette. Pour zoomer en avant il faut faire tourner la roulette vers le haut ou appuyer sur le bouton "Zoom +". Pour dézoomer il faut faire tourner la roulette vers le bas ou appuyer sur le bouton "Zoom -". Pour réinitialiser le zoom cliquez sur bouton "reset".

Selection

Les vues 3D représente les dispositifs réels. Grace à ces vue 3D vous pouvez sélectionner les capteurs dont vous voulez analyser les données. Pour sélectionner un capteur, il suffit de cliquer dessus. Pour sélectionner plusieurs capteurs, cliquez sur l'ensemble des capteurs que vous voulez sélectionner. Les capteurs sélectionnés s'ajoutent automatiquement au graphique et au tableau statistique. Pour désélectionner un capteur il suffit de recliquer dessus. À noter, les sondes sélectionnées sont listées sous la vue 3D.

Changer de vue

Pour passer d'une vue 3D à l'autre, il suffit de cliquer sur le bouton de la vue souhaitée. De base, la vue proposé est la vue 3D de la première installation, celle en face des amphithéâtres. Changer de vue désélectionne les sondes préalablement sélectionnées.

Le graphique

Le graphique permet de visualiser les données relatives aux capteurs sélectionné. La génération du graphique se fait automatiquement lors de la sélection des sondes. Autant de sondes souhaitées peuvent être ajoutées au graphique. Sur le graphique, courbe et capteur sont de la même couleur pour faciliter la lecture.

Definir un intervalle

Une fois des capteurs sélectionnés, il faut définir un intervalle de temps entre lequel on souhaite étudier les données. Par défaut, la semaine courante est sélectionnée. Pour modifier les dates, il suffit de cliquer dessus. Vous pouvez alors écrire une nouvelle date ou utiliser le calendrier pour sélectionner une date voulu.

Ajuster l'echelle

Pour faciliter l'analyse des données, il est possible d'ajuster l'échelle du graphique. Par défaut, l'échelle correspond à l'intervalle de dates précédemment choisi. Il est possible de le modifier avec les options de zoom. On peut sélectionner une abscisse par année, par mois, par jours, ou par heures.

Zoomer sur le graphique

Pour étudier les données sur un intervalle précis, il est possible de zoomer sur les courbes. Pour cela, il suffit de sélectionner l'intervalle souhaité avec le clic gauche de la souris (comme pour surligner un texte). Il est possible de préciser cette sélection ou de la déplacer grâce aux curseurs sous le graphique.

Exporter le graphique

Pour exporter le graphique, il faut utiliser le menu de droite. Vous pouvez imprimez le graphique ou l'exporter en PNG, en JPEG, en PGF, en SVG, en CSV ou en XLS.

Le tableau statistique

En plus du graphique, le tableau statistique permet d'observer les données de capteurs et surtout de les classer. Le tableau se rempli automatiquement lorsqu'on sélectionne un capteur et lorsque l'on modifie l'intervalle d'observation du graphique.

Trier le tableau

Le tableau liste la valeur minimale des données récupéré par chaque capteur, la valeur maximale, la moyenne des valeurs, et enfin l'écart-type. Il est possible de trier le tableau selon ces valeurs. Pour cela il faut cliquer sur le libellé de la valeur voulu. Cliquez une fois pour avoir un classement croissant et recliquez pour avoir un classement décroissant. Par défaut, les valeurs sont triées par ordre alphabétique selon le nom des capteurs.

L'administration

La partie administration permet de gérer l'ensemble de l'ensemble des sondes et capteurs connectés à la base de données.

Ajouter un capteur

Pour ajouter un capteur, il faut se rendre dans l'onglet "Gérer capteur" de l'administration. Pour pouvoir ajouter un nouveau capteur à la base de données, il faut lui renseigner un nom, le dispositif auquel il appartient, le type de capteur que c'est, l'unité de la valeur enregistré par le capteur et enfin les coordonnées du capteur.

Supprimer un capteur

Pour supprimer un capteur, il faut le sélectionner dans la liste des capteurs. Une fois sélectionné, il suffit de valider grâce au bouton supprimer. Il est possible de sélectionner plusieurs capteurs à supprimer en utilisant la touche "Crtl".

Ajouter un dispositif

Pour ajouter un capteur, il faut se rendre dans l'onglet "Gérer dispositif" de l'administration. Pour pouvoir ajouter un nouveau capteur à la base de données, il faut lui renseigner un nom, le type de dispositif que c'est (corbeille, sonde ...), renseigner le lieu où se trouve le dispositif et enfin les coordonnées du capteur.

Supprimer un dispositif

Pour supprimer un dispositif, il faut le sélectionner dans la liste des dispositifs. Une fois sélectionné, il suffit de valider grâce au bouton supprimer. Si des capteurs lui sont encore attachés, la suppression est refusée. Cependant, il est possible de supprimer tous les capteurs lié à un dispositif en cochant la case "Supprimer les capteurs liés aux dispositifs". Il est possible de sélectionner plusieurs dispositifs à supprimer en utilisant la touche "Crtl".

Ajouter une carte Arduino

Pour ajouter une carte arduino, il faut se rendre dans l'onglet "Arduino". Pour l'ajouter il faut lui renseigner un nom ainsi que l'adresse MAC de la carte Arduino à connecter.
